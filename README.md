## 작업 사항

* 노션을 통해 공유된 요구사항 명세 및 Figma 디자인 적용

* 시멘틱 마크업을 최대한 작성하였으며, 반응형 웹 구현을 위해 공유된 디자인 외 1280px, 720px 기준화면을 추가

* 스크롤 헤더의 경우 모든 해상도에서 고정으로 처리

* 모달창 및 모바일 해상도 hamburger menu 또한 구현 완료하였으며, 현재 주석처리한 상태
